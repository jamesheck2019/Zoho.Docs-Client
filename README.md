# Zoho.Docs-Client
a client to manage your files/folders in "Zoho.Docs" cloud

* software to manage your files in Zoho.Docs
* copy files folders
* move files folders
* rename files folders
* delete files folders
* upload download files folders
* share files folders
* explor & Download shared folder
